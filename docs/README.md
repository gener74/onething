# Imatges del README

El README principal espera aquests fitxers en aquesta carpeta. Afegeix-los abans
d'ensenyar el portfoli (ara els enllaços es veuran trencats fins que hi siguin):

| Fitxer | Què ha de mostrar | On capturar-ho |
| --- | --- | --- |
| `onething-demo.gif` | Recorregut curt (10-15 s): brain dump → Comença → Com et sents / Quant temps → primer pas → Fet. | Grava la pantalla i converteix-ho a GIF. |
| `onething-focus.png` | El Mode Focus amb UN pas a la vista + el cercle/compte enrere. | Captura a producció o `vercel dev`. |
| `onething-empty.png` | L'estat buit amb les fulles de sàlvia caient i la frase motivadora. | Buida totes les tasques i captura. |

Consells:
- Amplada recomanada ~1000-1200 px; format vertical (mòbil) queda molt bé per a una app de focus.
- Per al GIF, eines fàcils: ScreenToGif (Windows), o gravar amb el sistema i convertir.
- Manté el to calmat: poca cosa a pantalla, sense barres del navegador si pots.
